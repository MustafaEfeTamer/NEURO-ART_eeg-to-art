# NEURO-ART 🧠🎨

NÖRO-ART is an innovative Brain-Computer Interface (BCI) application that translates real-time human emotions into dynamic abstract generative art. By utilizing an Emotiv EEG headset, a trained Deep Learning model, and the Pollinations AI image generation API, NÖRO-ART captures your brainwaves, maps them to a 2D emotion plane (Valence and Arousal), and visualizes your mental state through stunning AI-generated paintings.

## 🚀 Features
- **Real-Time EEG Processing:** Connects to Emotiv headsets via the official Emotiv Cortex SDK.
- **Deep Learning Emotion Prediction:** Uses a custom trained TensorFlow/Keras model (`neuro_art_regression_tanh.keras`) to predict Valence (positivity/negativity) and Arousal (energy/calmness) from 14-channel EEG data.
- **Personal Baseline Calibration:** Features a 15-second calibration phase to establish the user's neutral state, dynamically scaling predictions for highly accurate personalization.
- **Generative Art Engine:** Maps the detected emotional quadrant to a crafted prompt, utilizing the **Pollinations AI** API to generate unique, high-resolution abstract art on the fly.
- **Dummy/Simulation Mode:** No EEG headset? No problem! The system automatically falls back to a simulation mode if a headset is not detected, generating random dummy data so you can still explore the dashboard and art generation.
- **Glassmorphic Modern UI:** A beautifully crafted, responsive web dashboard built with HTML, TailwindCSS, and Vanilla JavaScript.

## 🛠️ Technology Stack
- **Backend:** Python, Flask, TensorFlow / Keras, NumPy, Emotiv Cortex API, Python-Dispatch
- **Frontend:** HTML5, TailwindCSS
- **AI Generation:** [Pollinations AI](https://pollinations.ai) (Flux / Turbo models)

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/noro-art.git
   cd noro-art
   ```

2. **Install the required Python packages:**
   Make sure you have Python 3.8+ installed.
   ```bash
   pip install flask tensorflow numpy websocket-client python-dispatch
   ```

3. **Get your Pollinations API Key:**
   To enable the generative art engine, you need a free API key from Pollinations AI.
   - Go to [Pollinations AI](https://pollinations.ai) and obtain your API key.
   - Open the `code.html` file.
   - Find line `263` (approximate):
     ```javascript
     // 🔑 API CONFIGURATION
     const API_KEY = "YOUR_POLLINATIONS_API_KEY_HERE"; // Insert your key here
     ```
   - Replace `"YOUR_POLLINATIONS_API_KEY_HERE"` with your actual API key. 
   - **Do not commit your API key to public repositories!**

4. **Run the Application:**
   ```bash
   python app.py
   ```

5. **Access the Dashboard:**
   Open your browser and navigate to:
   👉 `http://127.0.0.1:5000`

## 🎧 Using with an Emotiv Headset
By default, the application attempts to connect to a running instance of the Emotiv Cortex app. Make sure your Emotiv headset (e.g., EPOC X) is turned on, connected via Bluetooth/Dongle, and the Emotiv App is open on your PC. 

If no headset is found within 5 seconds, NÖRO-ART seamlessly switches to **Dummy Mode**, allowing you to test the interface without hardware.

## 🧠 How it Works
1. **Brainwaves (EEG):** 14-channel raw data is streamed at 128Hz.
2. **Signal Processing:** Data is normalized (Z-score) and fed into a sliding window buffer.
3. **Inference:** The Keras model outputs 2D emotion coordinates (Valence & Arousal) from -1.0 to 1.0.
4. **Calibration (Tare):** The user's baseline state is subtracted to capture relative emotional shifts.
5. **Prompt Engineering:** The 2D coordinates are mapped to a specific artistic style (e.g., *High Valence, High Arousal* -> "abstract fluid art, explosion of vivid neon colors...").
6. **Visualization:** The prompt is sent to Pollinations AI, and the resulting masterpiece is displayed on the dashboard.
